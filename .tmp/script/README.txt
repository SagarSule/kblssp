importMobionProjectContent.ant
  - this script copies the content from a mobione project's www/<project> dir
    into the www/ directory of a PhoneGap project.
    
addPlugins
  - this is a list of commandline that will import the PhoneGap/Cordova plugins
    required to build a MobiOne project using the local PhoneGap SDK with
    the Android SDK and iOS XCode tools.
